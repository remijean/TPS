# Third-person controller

Third-person controller for Godot game engine.

![Screenshot](assets/screenshot.png)

## Features

- Looking around
- Walking/sprinting
- Jumping/rolling
- Crouching
- Aiming
- Zooming

## Keys

- Walk: `W/A/S/D`
- Sprint: `Left Shift`
- Jump: `Space`
- AIM: `Mouse Right Button`
- Roll: AIM+`Space`
- Crouch: `Control`
- Zoom: `Mouse Wheel`
- Capture mouse: `Esc`
- Toggle fullscreen: `Alt+Enter`
